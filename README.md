# NoteBook
一个在线笔记记录网站<br>
后台通过django接受数据，前台通过js实现交互<br>
没有使用django的页面渲染<br>
功能包括：<br>
* 用户注册（非注册用户也可快速记录笔记）
* 邮件提醒(暂未实现)
* 文章密码
* 搜索功能（有难度，未来考虑引入其他搜索引擎）
* 支持html标签效果<br><br><br>
